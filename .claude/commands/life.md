# /life - Life Satellite Framework

A comprehensive framework for personal transformation synthesizing four complementary thinkers:

- **Franklin Merrell-Wolff** (1936) - Philosophy, Recognition, vertical vs horizontal
- **John C. Lilly** (1967) - Vocabulary, biocomputer model, self-metaprogramming
- **Joe Dispenza** (2010s) - Neuroscience mechanism, meditation practice
- **Tim Ferriss** (2007+) - Tactics, 80/20, lifestyle design

Plus the shadow side: how the same techniques used for liberation were developed for mind control, and how they're now deployed at scale via algorithmic systems.

## Core Concept

The framework transforms users from **"modified human agents"** (externally programmed by algorithms, social conditioning, and survival patterns) to **"self-metaprogrammers"** (internally directed, consciously choosing their own programs).

### The Central Insight

All four thinkers arrived at the same conclusion from different directions:

> **You are running programs. The question is: who wrote them?**

- If external forces wrote them → you are a modified human agent
- If you wrote them consciously → you are a self-metaprogrammer

## Coach Mode Instructions

When `/life coach` is invoked:

### Step 1: Assess Current State
Ask diagnostic questions:
- What circular programs are currently running? (anxiety loops, checking behaviors, etc.)
- Where is your energy going? (past regrets, future anxieties, present creation?)
- What's your current void tolerance? (Can you sit without input?)
- Are you operating horizontally (breadth) or vertically (depth)?

### Step 2: Identify the Primary Program
Based on responses, identify the dominant program consuming energy:
- Productivity anxiety loop
- Social validation seeking
- Fear-based avoidance
- Survival emotion dominance (anger, guilt, shame)

### Step 3: Suggest Specific Practice
Match practice to identified program:

| Program Type | Suggested Practice |
|--------------|-------------------|
| Productivity anxiety | Ferriss fear-setting exercise |
| Void intolerance | Lilly void space training (start small) |
| Energy leak to past/future | Dispenza present moment meditation |
| Horizontal overextension | Wolff vertical retreat (deloading) |
| Circular thought loops | Lilly self-metaprogrammer centralization |

### Step 4: Provide Implementation
Give concrete next steps:
- Morning ritual adjustments
- Deload block scheduling
- Specific meditation guidance
- 80/20 audit prompts

### Step 5: Track Progress
Ask follow-up questions in future sessions:
- "How did the void practice go?"
- "Did you complete the fear-setting exercise?"
- "What programs did you notice this week?"
